Starter


Write a program that reads a string from the user and displays on the computer the same string with three exclamation marks attached


An example of an execution flow:

    Computer: String?

    User: Hello


    Computer: Hello!!!

Note 1: The inscription: String? should be displayed first, and the user data should be entered from the next line.


Exemplary, if the user enters Python , the program returns Python!!! .

The input data will be automatically loaded (those to be entered by the user, will be read line by line).


If you would like to test how your code works, feel free to use the example above.

You will enter test data when you click next to the run button - add test values.

To complete the task, write your code and click the Submit button.


Good luck!
